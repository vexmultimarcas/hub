# Sprint Visual Estrutural — Lista Compacta e Experiência Premium

## Objetivo

Concentrar ajustes relevantes em uma única Sprint, evitando microalterações e reduzindo o tempo de testes.

## Entregas

- Tela **Veículos** transformada em lista compacta para localização rápida.
- Nome do veículo aparece completo no desktop, evitando cortes desnecessários.
- No celular, o veículo fica em uma linha curta para economizar espaço.
- Detalhes completos continuam no painel lateral ao clicar no veículo.
- Filtros, pesquisa, drawer, status e permissões foram preservados.
- Relatórios receberam acabamento executivo leve em CSS puro.
- Nova Venda recebeu ajustes de respiro visual sem mudar IDs, campos ou salvamento.

## Regra de UX

A tela Veículos deve funcionar como catálogo rápido:

```txt
Ford Ecosport FLS
Volkswagen Gol 1.6 CL
Fiat Strada CL
```

As informações completas ficam apenas no detalhe do veículo.

## Áreas não alteradas

- Login
- Logout
- Firebase Authentication
- Inicialização Firebase
- Salvamento Firestore
- Regras de ADM / Usuário
- Perfil
- Painel de usuários
