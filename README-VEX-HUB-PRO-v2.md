# VEX HUB PRO v2.0 — Fase 01

Esta entrega inicia a versão PRO do VEX HUB com foco em Dashboard Executivo, Sidebar, Design System, cards e responsividade.

## Como testar

1. Faça backup da pasta atual do projeto.
2. Extraia este ZIP.
3. Abra o `index.html` ou publique no GitHub Pages.
4. Faça login com seu usuário Firebase.
5. Valide Dashboard, Nova Venda, Veículos e Relatórios.

## Observação

A configuração real do Firebase foi preservada a partir do projeto enviado.


## Fase 02 — Veículos Premium

Esta fase transforma o antigo Histórico em uma área de Veículos Vendidos com experiência premium.

### Como testar
1. Faça login.
2. Abra o menu Veículos.
3. Verifique os cards compactos.
4. Clique em um veículo.
5. Confira o drawer lateral com os dados completos.
6. Altere status e transferência.
7. Feche o drawer.
8. Cadastre uma nova venda e confirme se ela aparece na lista.

## Fase 03 BUILD 02

### Como testar
1. Faça login.
2. Abra Nova Venda.
3. Preencha todos os campos obrigatórios antigos.
4. Preencha FIPE, versão, câmbio, cor, placa e KM.
5. Salve a venda.
6. Confirme se aparece em Veículos.
